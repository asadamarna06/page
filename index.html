تخصص تصميم صفحات الويب

مقدمة عن المادة الادوات المستخدمة شرح عن الادوات مواقع البرامج المستخدمة و طريقة التحميل مواقع اخرى التواصل

مقدمة عن المادة

مادة "تصميم صفحات ويب" تهدف إلى تعليم الطلاب إنشاء صفحات ومواقع إنترنت باستخدام لغات الويب الأساسية (HTML، CSS، JavaScript) بالإضافة إلى تعلّم الأدوات المساعدة في التصميم، التطوير، وإدارة النسخ. سنتدرّب على بناء صفحات ثابتة وتفاعلية، ونقدّم مبادئ التصميم المتجاوب وتجربة المستخدم.

ما سنغطيه

  • هيكلة الصفحة بـ HTML
  • تصميم وتخطيط باستخدام CSS و Flex/Grid
  • إضافة تفاعل بواسطة JavaScript
  • استخدام مكتبات مثل Bootstrap (اختياري)
  • أدوات تطوير: محررات، DevTools، Git

مهارات متوقعة

  • بناء صفحات متجاوبة تعمل على الجوال والكمبيوتر
  • التعامل مع نظام تحكم بالإصدارات
  • قراءة تصميم UI وتحويله لواجهة فعلية
  • تصحيح الأخطاء وتحسين الأداء

الهدف

  • التعرف على مفهوم صفحات الويب وأهميتها في العصر الرقمي.
  • فهم أساسيات تصميم صفحات الويب ومكوناتها (النصوص، الصور، الروابط).
  • اكتساب مهارة إنشاء صفحة ويب بسيطة باستخدام الأدوات أو اللغات المناسبة.
  • تنمية القدرة على تنظيم المحتوى بشكل واضح وجذا

مكان العمل

  • التعرف على مفهوم مكان العمل وأهميته في الحياة المهنية.
  • اكتساب مهارات التنظيم والترتيب داخل مكان العمل.
  • الالتزام بقواعد السلامة والأمان في بيئة العمل.
  • تنمية مهارات التعاون والعمل ضمن فريق

البرامج والأدوات المستخدمة

فيما يلي قائمة شائعة بالأدوات التي تستخدم في هذه المادة مع شرح مبسط لكل منها.

HTML (لغة توصيف النصوص)

الأساس الذي من خلاله نُنشئ بنية الصفحة: عناوين، فقرات، روابط، صور، جداول، ونماذج. يتعلّم الطالب علامات HTML وكيفية بناء صفحات منظمة وسليمة دلالياً (semantic).

CSS (تصميم وتخطيط)

تُستخدم لتنسيق المظهر: الألوان، خطوط، تخطيطات (Flexbox, Grid). سنتعلم بناء قوالب متجاوبة (responsive) وتحسين تجربة المستخدم.

JavaScript

لغة برمجة على جانب العميل تُضيف تفاعلًا للصفحات: التحقق من النماذج، التلاعب بالعناصر DOM، استدعاءات إلى خوادم، وإنشاء تطبيقات بسيطة من جهة العميل.

Visual Studio Code

محرر شيفرة مجاني وشائع يدعم HTML/CSS/JS مع إضافات مساعدة (Emmet، Live Server، Git integration) لتسريع التطوير.

Git و GitHub

نظام للتحكم بالإصدارات يتيح تتبّع التغييرات والعمل التعاوني. GitHub يقدم مستودعات على الإنترنت لمشاركة المشروع واستضافة الصفحات (GitHub Pages).

Photoshop / أدوات تحرير الصور

تُستخدم لتحرير الصور، إنشاء شعارات، وتجهيز الأصول (assets) للاستخدام في تصميم الويب. يمكن استبدالها بأدوات مجانية مثل GIMP أو Photopea.
